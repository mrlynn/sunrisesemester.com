## Project brief

Sunrise Semester is a Next.js App Router site for an AA meeting group. It includes public pages (meetings, resources, stories, reflections) and a password-protected admin area for editing content.

### Goals
- Provide a reliable, accessible public website for the group.
- Allow simple editorial management via `/admin` without a full user system.
- Persist site content in MongoDB Atlas via Mongoose models and API routes.
